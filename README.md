# Terence McKenna Corpus

A structured knowledge base of Terence McKenna's teachings, theories, and intellectual legacy — organized for use by AI systems, image-prompt generators, shader-art tools, and contextual apps.

---

## What This Is

This repository contains ~200 analytical markdown files forensically extracted from two primary source documents:

- **ghost terence language.pdf** — McKenna's philosophy of language, visible speech, and the evolution of communication
- **ghost terence time wave zero.pdf** — McKenna's Timewave Zero theory, novelty mathematics, and eschatological framework

Each concept is broken into five analytical depth layers, cross-referenced, and supplemented with thematic essays, critical residue files, and meta-navigation documents.

## File Naming Depth Layers

Each topic in `language/` and `timewave/` has **five depth-layer files**:

| Suffix | Layer | What It Contains |
|---|---|---|
| `_overview_surface.md` | Surface | High-level summary of the topic |
| `_origins_deep.md` | Deep | Historical & biographical roots |
| `_analysis_middle.md` | Middle | Analytical breakdown of the theory |
| `_implications_marrow.md` | Marrow | Philosophical & practical implications |
| `_legacy_residue.md` | Residue | Influence, critique, and unresolved threads |
